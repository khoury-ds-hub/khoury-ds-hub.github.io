About the Data Sessions

2020-02-07 11:30

Starting on February 5th, we are proud to introduce the beginning of our new revamped Data Science Sessions! Where we had previously designed our Data Science Sessions as a sort of free-form project night, we are now reimagining our sessions as a lecture series designed to introduce topics in Data Science that may prove useful for all Northeastern University students interested in picking up some ancilliary skills. Possible topic areas include overviews of machine learning algorithms, introductions to relevant programming languages, and crash courses on useful tools such as Git and GitHub. Listed below is a tentative list of possible topics, which is still subject to change, so stay on the lookout for updates to this post!

And for those of you worried about project nights, don't worry! We're still vested in bringing you project events, so stay tuned for new updates!

  • Sessions are held on a biweekly basis:
    • Room: WVH 366
    • Dates: Jan. 22, Feb. 5, Feb. 19, Mar. 4, Mar. 18, Apr. 1
    • Time: 6:00 - 8:00 p.m.
  • February 5th:
    • Gradient Descent
    • Logistic Regression
  • February 19th:
    • Neural Networks
    • TBD: How to use GitHub
  • March 4th:
    • Perceptron & Support Vector Machines
  • March 18th:
    • Decision Trees & Ensemble Learning
  • April 1st:
    • TBD: Deep Neural Networks, Large-Scale Parallel Processing, or Relational Databases
